# Agent 起跑线：77 天 Python → Hello-Agents 学习计划

面向浅学过 C/Java、正在迁移到 Python 的学习者。每天约 90 分钟：前 21 天集中掌握 Python 特有语法与工程基础，后 56 天沿 Datawhale《从零开始构建智能体》主线完成理论、经典范式、自研框架、高级能力和毕业项目。

## 使用

直接用浏览器打开 `index.html`。页面不依赖后端，打卡进度、子任务与每日复盘会保存在浏览器的 `localStorage` 中。

建议先设置自己的 Day 1 日期，然后每天只完成当天的三个动作：

1. 学习约 25 分钟；
2. 动手编码约 45 分钟；
3. 测试、复盘约 20 分钟。

你已有 C/Java 基础：遇到熟悉的条件、循环、类等概念，不必重复听长课。直接完成当天产物和边界测试；能独立写出并解释，就进入下一天。

页面支持：

- 77 天逐日任务与验收目标
- 每日子任务打卡
- 总进度、连续天数、投入时间统计
- 按当前周或未完成任务筛选
- 每日调试复盘
- JSON 格式导入、导出进度

## 路线依据

- 官方教程：<https://github.com/datawhalechina/hello-agents>
- 第 1—3 周：从 C/Java 迁移到 Python，重点练容器、动态类型、函数参数、文件/JSON、异常、测试和环境管理。
- 第 4—11 周：依次对应教程第 1—16 章；第 11 章 Agentic RL 以理解流程为主，不要求当前阶段投入算力训练。
- 每章都以可运行代码、测试/评估结果或架构复盘作为完成证据，不能只以“读完”打卡。

## 学习原则

- 每天必须留下可运行代码或调试记录，不以“看完视频”作为完成标准。
- 遇到错误先读 traceback，再搜索；记录“现象、原因、修复”。
- 第 4 周起的大模型 API 练习可使用真实 Key，也可以用固定 JSON Mock，不让费用或账号阻断进度。
- 每周第 7 天不追新知识，只重写、测试与复盘。
