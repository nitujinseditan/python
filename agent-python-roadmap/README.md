# Agent 起跑线：56 天 Python → Agent 学习计划

面向 Python 完全零基础学习者的逐日训练路径。每天约 90 分钟，从语言基础、调试和测试一路走到能独立实现一个小型 Agent。

## 使用

直接用浏览器打开 `index.html`。页面不依赖后端，打卡进度、子任务与每日复盘会保存在浏览器的 `localStorage` 中。

建议先设置自己的 Day 1 日期，然后每天只完成当天的三个动作：

1. 学习约 25 分钟；
2. 动手编码约 45 分钟；
3. 测试、复盘约 20 分钟。

页面支持：

- 56 天逐日任务与验收目标
- 每日子任务打卡
- 总进度、连续天数、投入时间统计
- 按当前周或未完成任务筛选
- 每日调试复盘
- JSON 格式导入、导出进度

## 学习原则

- 每天必须留下可运行代码或调试记录，不以“看完视频”作为完成标准。
- 遇到错误先读 traceback，再搜索；记录“现象、原因、修复”。
- 第 5 周的大模型 API 练习可使用真实 Key，也可以用固定 JSON Mock，不让费用或账号阻断进度。
- 每周第 7 天不追新知识，只重写、测试与复盘。

## 数据说明

所有数据只保存在当前浏览器。清理浏览器站点数据前，请先用“导出进度”备份。
